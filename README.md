# calendar-scrapbox-sync

Google Calendar と Scrapbox を連携し、
研究室運営OSの時間軸と知識管理を統合するためのツールです。

## Goal

Google Calendar
↓
Scrapbox Project Page
↓
Task / Log / Knowledge

## Version 0.1

- Google Calendarイベント取得
- Scrapbox Projectリンク生成
- Google Calendar説明欄へProjectリンクを追記
- Dry Run対応
